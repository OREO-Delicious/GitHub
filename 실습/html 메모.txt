문법

<!DOCTYPE html> : 시작하기 전에 무조건 써야하는 문법
 
<~> 내용 </~> : 첫 번째가 시작 두 번째가 끝(태그 들)
태그 안에 들어가는 것들은 속성들임 속성은 순서가 상관 없음
ex) href , title, target ... 

<h1>내용 </h1> : 제목 크게 하는거 
<h2>내용 </h2> : 제목 작게 하는거 
<storng>내용 </strong> : 내용 강조하는거
<a href = "사이트 링크"> 이름 </a> : 사이트 링크 넣는거( 이렇게 하면 "이름" 클릭시 사이트로 이동함)
<a href = "사이트 링크" target = "_blank"> 이름 </a> : 이름을 클릭하면 사이트가 새창으로 나옴
title = "내용" : 마우스 커서를 갖다 대면 "내용" 이름이 미리보기로 나옴
<a> : a태그 중요함 

리스트 태그의 중첩
목록을 나눌 때 특수한 기호를 이용하는데 
이 때 사용하는 태그
<li> : list 의 약자

항목들의 성격을 다르게 하고 싶으면 <ul>을 사용함 그룹핑 : unordered list
ex)
	<ul>
	과목
	</ul>
	
	<ul>
	이름
	</ul>
순서가 있는 목록을 만들고 싶다면 <ul> 대신 <ol>사용 앞에 숫자가 붙는다 : ordered list

웹 이름을 기가막히게 쓰고 싶을 때 쓰는 태그 
<title>

글씨가 깨질 때 쓰는 태그

<meta charest = "utf-8">

<title> 이나 utf 는 본문에 해당되는 것이 아님 문서를 꾸며주는 역할을 함 그래서 본문과 본문이 아닌것은 나눠야 함, head 태그와 BODY 태그는 html태그로 감싸야 함
<html>
 <head> 본문 아닌것 
 <body> 본문 
</html>

항목을 입력하면 항목으로 이동하는 링크를 만들고 큰 제목을 (ex 네이버 클릭시 네이버로 이동) 클릭하면 원래 화면으로 돌아오게 만들기
